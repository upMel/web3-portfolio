# Frontend — Setup

## Stack

- **Next.js 14** (App Router)
- **Tailwind CSS** (styling)
- **wagmi v2** (React hooks for Ethereum)
- **viem** (low-level Ethereum interactions)
- **RainbowKit** (wallet connection UI)

## Setup Commands

```bash
cd frontend
npm install
npm run dev
```

## Key Pages

| Route | Description |
|---|---|
| `/` | Home / landing page |
| `/portfolio` | Lists all portfolio projects |
| `/project/[id]` | Single project detail page |
| `/admin` | Owner-only: add/edit projects |
